omdb-cli
========

A command line tool for The Open Movie Database API ( `omdb`_), powered by `omdb.py`_

Usage
_____

Search in omdb

::

    omdb -s 'A new hope'

Search by imdb ID

::

    omdb -i tt0251413

Search by title

::

    omdb -t 'mad max'

.. _omdb.py: https://github.com/dgilland/omdb.py
.. _omdb: http://www.omdbapi.com/
